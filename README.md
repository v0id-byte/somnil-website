# Somnil 产品官网

> 床头睡眠环境被动干预 — somnil.top

静态官网（HTML + CSS + Vanilla JS），展示 Somnil 睡眠焦虑检测设备产品信息。

---

## 技术栈

- **前端**: 原生 HTML5 + CSS3 + JavaScript（无框架）
- **字体**: Inter（Google Fonts）
- **部署**: GitHub Pages

---

## 功能列表

- 🌙 **产品介绍**：Somnil 设备功能、原理、FAQ
- 📈 **临床数据展示**：睡眠分期 AUC 0.936 等核心指标
- 📧 **联系表单**：隐私政策、服务条款页面
- 🔗 **社媒矩阵**：小红书内容同步展示

---

## 安装说明

```bash
# 克隆项目
git clone https://github.com/v0id-byte/somnil-website.git
cd somnil-website

# 本地预览（任意静态服务器）
python3 -m http.server 8080
# 访问 http://localhost:8080
```

> 部署到 GitHub Pages：推送至 `main` 分支后自动生效，CNAME 配置 `somnil.top`。
